# Contributing to Employee Leave Management System

Thank you for your interest in contributing to this project! Here’s how you can get started:

## 📌 Getting Started
1. **Fork the repository** on GitHub.
2. **Clone your fork** locally:
      git clone (https://github.com/ReadNRevise/Intern-Hiring-React.git)

Create a branch for your work:
    git checkout -b intern-your-name

Follow the setup instructions in the README.md.

Submitting Your Work

Commit your changes with meaningful messages:

 git add .
 git commit -m "feat: add user authentication"

Push your branch to GitHub:
 git push origin intern-your-name

Open a Pull Request to the main repository:

Go to the [original repository] https://github.com/ReadNRevise/Intern-Hiring-React.git on GitHub.

Click **Pull requests** > **New pull request**.

Select your branch and submit the PR with a clear title and description.

Guidelines

Write clean, readable, and modular code.
Add comments to explain complex logic.
Test your code thoroughly before submitting.

Follow the project’s coding style and conventions.

Need Help?

Reach out to the team on email if you have any questions!
